Getting Started

For my first report, I’ll be using Netflix’s TuDum Top Ten data to uncover recent trends. This data is divided into two sets of data:

  • Global Top 10
  • Country-wide Top 10

Let’s get started by downloading the data.

format_titles <- function(df){
  colnames(df) <- str_replace_all(colnames(df), "_", " ") |> str_to_title()
  df
}

# Single pipeline with all transformations
GLOBAL_TOP_10 |>
